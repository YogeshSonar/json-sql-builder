    SELECT Statements

    To query the database using a select statement you have to take the $select operator.

    The minimum $select must supply the $columns operator, to specify at least one column. Optionally you can use:

    • $distinct
    • $all
    • $from
    • $where
    • $groupBy
    • $having
    • $sort
    • and all operators from your specific language dialect

    Just a short example:

    $select: {
        $columns: [
            'first_name',
            { first_name_count: { $count: '*' } }
        ],
        $from: 'people',
        $groupBy: ['first_name'],
        $having: {
            $expr: { $count: '*', $gt: 2 }
        }
}

    UNION SELECT Statements

    To create a query using the UNION clause you have to use the $union operator and define an Array of $select-Objects. Check the Syntax and Examples.

    Example

    var query = sqlbuilder.build({
    $union: [{
        $select: {
            $from: 'people',
            $columns: ['first_name', 'last_name'],
            $where: { id: 1 }
        }}, {
        $select: {
            $from: 'people_history',
            $columns: ['first_name', 'last_name'],
            $where: { id: 1 }
        }
    }]
});
