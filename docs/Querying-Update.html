    UPDATE Statements

    To query the database and update existing records you have to use the $update operator. Check the Syntax and Examples.

    Example

    $update: {
    $table: 'people',
    $set: {
        first_name: 'John',
        last_name: 'Doe',
    },
    $where: { age: { $gte: 18 } }
}
